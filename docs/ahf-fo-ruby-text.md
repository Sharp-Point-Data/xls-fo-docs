# axf:ruby-text

## Summary

Contains the annotation characters for the base characters in the preceding `axf:ruby-base` object. This element is a child of `axf:ruby` and must follow an `axf:ruby-base` element to form a ruby pair.

See `axf:ruby` for full usage context and examples.

## Properties

None.

## Parent Objects

- `axf:ruby`

## Child Objects

Inline content (the annotation/pronunciation characters).
