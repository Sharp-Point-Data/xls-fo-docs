# Antenna House Extension Properties: Ruby Properties

<!-- Source: XSL-FO-Reference-74-MID.pdf pp.312-476 -->

Properties for ruby annotation typesetting, controlling ruby alignment, position, overhang, font, and sizing. These properties are applied to the axf:ruby extension formatting object. Note: Most ruby properties are documented in the axf:ruby formatting object documentation.

## axf:ruby-color

The use of the Ruby extension is documented comprehensively and with examples in the Ruby Extension section of the AHF Online Manual. This property controls the color of ruby annotation text.

| Attribute | Value |
|---|---|
| **Applies to** | axf:ruby |
| **Values** | &lt;color&gt; |
| **Initial** | (see axf:ruby documentation) |
| **Inherited** | (see axf:ruby documentation) |

Note: The remaining ruby-related properties (axf:ruby-align, axf:ruby-position, axf:ruby-overhang, axf:ruby-font-size, etc.) are covered in the `ahf-fo-ruby.md` formatting object documentation.
