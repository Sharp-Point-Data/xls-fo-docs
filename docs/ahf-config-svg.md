# Antenna House Configuration: SVG Output Settings

<!-- Source: XSL-FO-Reference-74-MID.pdf p.273 -->

## Overview

The `<svg-settings>` element configures SVG output for AH Formatter. It controls image handling, compression, output format, and file organization for SVG output. For additional details, see SVG Output in Output Formats of the Online Manual.

## Element: `<svg-settings>`

**Parent:** `<formatter-config>`

**Child elements:** none

## Attributes

### `copy-image-path`

Specifies the path where copies of images are stored in SVG output.

- **Applies to:** `<svg-settings>`

### `copy-image-prefix`

Specifies the prefix for copied image filenames.

- **Applies to:** `<svg-settings>`

### `create-copy-image-path`

Specifies whether to create the directory for copied images if it does not exist.

- **Applies to:** `<svg-settings>`

### `format`

Specifies the output format for SVG.

- **Applies to:** `<svg-settings>`

### `gzip-compression`

Specifies whether to use gzip compression for SVG output (producing SVGZ files).

- **Applies to:** `<svg-settings>`

### `image-conversion`

Specifies the image conversion method for SVG output.

- **Applies to:** `<svg-settings>`

### `image-processing`

Specifies the image processing method for SVG output.

- **Applies to:** `<svg-settings>`

### `jpeg-quality`

Specifies the JPEG quality for image output within SVG.

- **Applies to:** `<svg-settings>`

### `rename-copy-image`

Specifies whether to rename copied images in SVG output.

- **Applies to:** `<svg-settings>`

### `singlefile`

Specifies whether to output all pages into a single SVG file.

- **Applies to:** `<svg-settings>`

### `singlefile-number`

Specifies the numbering scheme when outputting multiple SVG files.

- **Applies to:** `<svg-settings>`

### `rasterize-resolution`

Specifies the resolution for rasterizing content in SVG output.

- **Applies to:** `<svg-settings>`

### `svg-version`

Specifies the SVG version for output.

- **Applies to:** `<svg-settings>`
