# axf:ruby-base

## Summary

Contains the base characters that are annotated by the corresponding `axf:ruby-text` object. This element is a child of `axf:ruby` and must be followed by an `axf:ruby-text` element to form a ruby pair.

See `axf:ruby` for full usage context and examples.

## Properties

None.

## Parent Objects

- `axf:ruby`

## Child Objects

Inline content (the base characters to be annotated).
