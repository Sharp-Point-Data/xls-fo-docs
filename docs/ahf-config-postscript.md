# Antenna House Configuration: PostScript Output Settings

<!-- Source: XSL-FO-Reference-74-MID.pdf p.273 -->

## Overview

The `<ps-settings>` element configures PostScript output for AH Formatter. It controls font embedding, transparency handling, and local file link behavior for PostScript output. For additional details, see PostScript Output in Output Formats of the Online Manual.

## Element: `<ps-settings>`

**Parent:** `<formatter-config>`

**Child elements:** none

## Attributes

### `noembed-font`

Specifies fonts that should not be embedded in the PostScript output.

- **Applies to:** `<ps-settings>`

### `transparency`

Controls the handling of transparency in PostScript output.

- **Applies to:** `<ps-settings>`

### `use-launch-for-local-file`

Specifies whether to use launch actions for local file links in PostScript output.

- **Applies to:** `<ps-settings>`
