# fo:index-page-number-prefix

## Summary

The `fo:index-page-number-prefix` formatting object specifies a static prefix for the cited page items created by `fo:index-key-reference`.

<!-- Source: https://www.w3.org/TR/xslfo20/#fo_index-page-number-prefix -->

## Areas

The `fo:index-page-number-prefix` formatting object does not directly produce any areas. Its children will be retrieved and used by `fo:index-page-citation-list` when formatting cited page items and cited page item ranges.

## Constraints

None.

## Content Model

<!-- Source: https://www.w3.org/TR/xslfo20/#fo_index-page-number-prefix -->
```xml
(#PCDATA|%inline;)*
```

## Properties

This formatting object has no specific properties beyond those inherited from the content model.

## Usage Notes

- The `fo:index-page-number-prefix` is an optional child of `fo:index-key-reference`. Its content is prepended to each page number generated by the parent `fo:index-key-reference`.
- A common use is to add brackets or other decorations around page numbers for specific types of index entries (e.g., "[" prefix and "]" suffix for figure references).
- The prefix areas have `keep-with-next.within-line` set to "always" to prevent line breaks between the prefix and the page number.
