<entry>
Any number of this element can be specified and is evaluated in order.
If the setting is the same, latter one will take precedence. The default
value is the one when the new unregistered operator is specified. When
the operator is already registered, omitted attribute values will not
be changed. <entry> is disregarded when src of <operator-dictionary> is
specified.
